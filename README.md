# CJA101G3 島遊Kha

## 專案簡介
在規劃台灣旅遊時，複雜的行程規劃常常令人感到困擾，獨自旅行時也可能面臨難以找到旅伴的窘境，旅途中的心得與攻略缺乏合適的分享平台。因此，
我們打造一個專屬於台灣旅遊愛好者的交流平台，匯集多元的行程資訊，提供旅人編輯行程、發起揪團旅遊、採購旅遊票卷，盡情分享心得與攻略的空間。
